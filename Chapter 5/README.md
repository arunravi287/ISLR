### Chapter 5 - Resampling Methods

Resampling Methods involve repeatedly drawing samples from a training set and refitting a model on each sample in order to obtain additional information about the fitted model. This chapter explores the following major topics:
1. Cross-Validation
2. Validation Set Approach
3. LOOCV
4. k-Fold Cross-Validation
5. The Bootstrap
